# C++ Equipment Management System

A C++ console-based system for managing equipment records. This project was developed to practice data structures, file handling, and searching/sorting algorithms while simulating a real-world equipment management application.

## Features
### Current Features
- Add equipment
- Display equipment

### Planned Features
- Search equipment
- Sort equipment
- Save data to file
- Edit equipment
- Delete equipment

## Technologies Used
- C++
- Visual Studio 2022
- Git & GitHub
- File Handling
- Arrays / Structures
- Sorting & Searching Algorithms
